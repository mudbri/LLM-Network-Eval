The file "fill_missing_values" fills the missing values given in the second folder with the values in the first folder. This is to remove nan values that students might have missed: python3 fill_missing_values.py Subcategories/Copies/Original-copy/5 Subcategories/Copies/Finished_relabelled_copy/5

Validating files is used to standardize all categories. This is necessary in order to compute the kappa scores

calculate_kappa.py calculates kappa scores between two folders

calculate_differences calculates the differences in values between two files
